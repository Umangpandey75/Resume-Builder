# Resume ATS Auditor — Frontend 🖥️

This directory contains the user interface and client-side logic for the Resume ATS Auditor application built with **React**, **Vite**, and **Tailwind CSS**.

For full-stack documentation, installation guides, and architecture diagrams, please refer to the main [Root README](../README.md).

## 🚀 Quick Start
1. Ensure dependencies are installed:
   ```bash
   npm install
   ```
2. Set up your `.env` variables (refer to `.env` file template):
   ```env
   VITE_DEFAULT_PROVIDER=gemini
   VITE_DEFAULT_GEMINI_API_KEY=your_key_here
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
